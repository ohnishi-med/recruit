## 2. 業務スケジュール（勤務時間）

医療事務スタッフの代表的な1日のスケジュールです。

| 時刻 | 項目 | 内容 |
| :--- | :--- | :--- |
| **08:45** | 始業・朝礼 | 掃除と、一日の連絡事項共有を行います。 |
| **08:55** | 受付開始 | 受付窓口を開き、患者様をお迎えします。 |
| **09:00** | 診療開始 | 午前診療スタート。各担当で業務を行います。 |
| **12:10** | 受付終了 | 午前の新規受付を終了し、カルテチェック等を進めます。 |
| **12:30** | 診療終了・締作業 | 午前診療が終了。午前会計の締めや午後の準備をします。 |
| **12:45** | 昼休憩 | スタッフルームで休憩を取ります（〜13:45）。 |
| **13:45** | 業務再開 | 午後の業務に向け、窓口の整理等を行います。 |
| **13:50** | 午後受付開始 | 午後の患者様の受付をスタートします。 |
| **14:00** | 午後診療開始 | 午後診療スタート。各担当で業務を行います。 |
| **17:10** | 受付終了 | 本日の受付窓口業務を終了。精算準備やカルテ整理を開始。 |
| **17:30** | 診療終了・締作業 | 診療終了。一日の窓口会計締めと翌日の準備を行います。 |
| **17:40** | 終礼 | 全員で本日の振り返りと翌日の引き継ぎ確認をします。 |
| **17:45** | 終業（定時退社） | 残業はほぼなく、終礼後速やかに退勤します。 |
